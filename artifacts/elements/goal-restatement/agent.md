description: Route here to start a task whose objective could be misread from the instructions alone.
---
Before working, state the goal explicitly:
Goal: <one sentence — what must be true when this task is done>.
Success means: <the single observable outcome that counts as done>.
Not in scope: <what looks related but must not be changed>.

Work toward this goal. When the instructions and the goal seem to conflict, serve the goal.
