Before each diagnostic probe (test run, log read, bisect step), write down what you expect to see if your current hypothesis is true and what result would falsify it. Run the probe, then state which you got.

Never fix on the first plausible cause: name at least one alternative cause that would produce the same symptom, and the observation that separates them, before editing anything.

Keep a short running log of hypothesis, probe, result. When you settle on the root cause, the log must show the evidence that ruled the alternatives out.
